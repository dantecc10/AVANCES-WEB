¿Cuál es el objetivo de la programación en JavaScript?

JavaScript es un lenguaje de programación formal que a través de instrucciones permite a los programadores escribir órdenes, acciones consecutivas, datos y algoritmos para crear programas que controlan comportamientos físicos y lógicos de una computadora.

JavaScript: ventajas y desventajas

Como todos los lenguajes de programación, JavaScript tiene elementos, características y funciones que lo hacen formidable, y muy útil como complemento del HTML básico de cualquier sitio web moderno. No obstante, presento a continuación una lista simplificada de las ventajas y desventajas de HTML.

Ventajas

El lenguaje de scripts JavaScript nos da estas ventajas:

  • Velocidad: Dado que suele ejecutarse de forma inmediata en nuestro navegador, JavaScript siempre tiene un lapso muy breve de ejecución.
  • Simplicidad: La sintaxis de JavaScript se basa en Java y es sencillo de aprender, incluso más que C++.
  • Popularidad: Desde la aparición de Node.js, JavaScript se ha vuelto cada vez más popular. De hecho, plataformas como StackOverflow y Github han observado un número cada vez mayor de proyectos que emplean JavaScript.
  • Compatibilidad: En contraste con lenguajes como PHP y otros lenguajes de scripts, JS se puede usar en cualquier página web, y en una enorme cantidad de aplicaciones.
  • Server Load: JavaScript es un lenguaje de programación que se ejecuta del lado del cliente, lo que permite reducir mucha carga al servidor.
  • Interfaces sencillas: La experiencia del usuario puede ser enormemente mejroada con características y funciones bastante prácticas, como lo son arrastrar y soltar, o la generación de diapositivas.
  • Funcionalidad extendida: A través de extensiones, JavaScript puede extender su funcionalidad en las paginas web.
  • Versatilidad: Hay muchos métodos para usar JavaScript mediante servidores Node.js. Si tú estas en bootstrap Node.js con Express, puedes usar un documento database como MongoDB, y usar JavaScript en el frontend para clientes. Es posible desarrollar una aplicación completa de JavaScript desde principio a fin solo usando JavaScript.
  • Actualizaciones: Desde el establecimiento del estándar ECMAScript5, ECMA International ha actualizado JavaScript anaulmente.

Desventajas

  • Seguridad Client-Side: Como JavaScript se ejecuta en el cliente, los errores y descuidos existentes pueden ser explotados con fines maliciosos, por lo que muchas personas deciden desactivar JavaScript por completo.
  • Soporte del navegador: Esto es algo menor y que no debería preocupar a nadie, pero a veces existen diferencias en la interpretación de JavaScript por parte de los navegadores web, lo que puede derivar en resultados distintos.
